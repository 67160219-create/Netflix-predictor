#  Netflix Availability Predictor

**Netflix Predictor** คือเว็บแอปพลิเคชันที่ใช้เทคโนโลยี Machine Learning ในการทำนาย "โอกาสที่ภาพยนตร์หรือซีรีส์เรื่องหนึ่งๆ จะมีฉายบนแพลตฟอร์ม Netflix" โดยวิเคราะห์จากสถิติและคะแนนวิจารณ์ต่างๆ 

แอปพลิเคชันนี้ออกแบบมาให้มี User Interface (UI) ที่ใช้งานง่าย ไม่ว่าผู้ใช้จะมีความรู้ด้าน Machine Learning หรือไม่ ก็สามารถเข้าใจและใช้งานได้ทันที

🔗 **https://netflix-predictor-lgeyegjcqmurvg69artje7.streamlit.app/**

##  ฟีเจอร์เด่น (Features)
แอปพลิเคชันนี้ถูกพัฒนาขึ้นโดยคำนึงถึงประสบการณ์ของผู้ใช้และความถูกต้องของข้อมูล:
- **User-Friendly UI:** ดีไซน์สไตล์ภาพยนตร์ ใช้งานง่ายด้วยระบบคลิกและเลื่อน (Sliders)
- **Input Validation:** มีการจำกัดขอบเขตของข้อมูล (เช่น คะแนน IMDb 0-10, ปีที่ฉายไม่เกินปีปัจจุบัน) เพื่อป้องกันผู้ใช้กรอกข้อมูลที่ไม่สมเหตุสมผล
- **Confidence Score:** แสดงผลการทำนาย (YES/NO) พร้อมบอก **ความน่าจะเป็น (Probability)** เป็นเปอร์เซ็นต์ (1-100%) เพื่อให้ผู้ใช้เข้าใจระดับความมั่นใจของโมเดล
- **Model Selection:** ผู้ใช้สามารถเลือกสลับใช้อัลกอริทึมทำนายได้ถึง 4 แบบ (Gradient Boosting, Random Forest, SVM, Logistic Regression)

##  คำอธิบายตัวแปร (Feature Descriptions)
โมเดลของเราใช้ข้อมูลตัวแปรหลัก 4 ตัวในการวิเคราะห์และทำนายผล ดังนี้:
1. **Year (ปีที่ฉาย):** ปี ค.ศ. ที่ภาพยนตร์หรือซีรีส์เรื่องนั้นออกฉายครั้งแรก (ยิ่งเป็นหนังใหม่ โอกาสที่ Netflix จะซื้อลิขสิทธิ์อาจแตกต่างจากหนังเก่า)
2. **Age Rating (เรทอายุ):** ระดับความเหมาะสมของผู้รับชม (เช่น 7+, 13+, 18+, all) ซึ่งมีผลต่อกลุ่มเป้าหมายของแพลตฟอร์ม
3. **IMDb Score (คะแนน IMDb):** คะแนนโหวตจากผู้ชมทางบ้านบนเว็บไซต์ IMDb (คะแนนเต็ม 10) สะท้อนถึงความนิยมโดยรวม
4. **Rotten Tomatoes Score (คะแนนมะเขือเทศเน่า):** คะแนนวิจารณ์จากนักวิจารณ์ภาพยนตร์ (คะแนนเต็ม 100%) สะท้อนถึงคุณภาพในมุมมองของนักวิจารณ์
